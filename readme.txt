Пример примитивного блога.

Были созданы следующие привилегии для различных групп пользователей: создание статьи, её просмотр, редактирование и удаление + удаление комментариев.

1. Незарегистрированный пользователь имеет возможность просматривать статьи, писать комментарии.

2. Зарегистрированный пользователь имеет возможность создавать статьи, просматривать их и писать комментарии.

3. Модератор имеет право удалять комментарии и радактировать статьи. Удалять статьи он не может.

4. Администратор имеет все права.
